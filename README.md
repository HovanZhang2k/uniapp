## Vlog日记应用

Vlog日记应用是一款基于uni-app开发的个人日记工具，特点包括：

- **多媒体支持**：支持文字、图片和视频内容
- **用户认证**：完整的登录/注册系统
- **日记管理**：创建、编辑、查看和删除日记
- **日期导航**：直观的日历界面
- **响应式设计**：适配不同设备屏幕

该项目已完成前端部分的开发，并提供了完整的后端API接口需求文档，方便与实际后端服务对接。
